Interview · First-Turn Fix + Prompt-on-Screen

Sửa bug miss audio đầu khi Simli avatar bật + UX hiển thị đề bài trong session để learner phản xạ tốt hơn.

Phân tích bug

Race window giữa START WS message (set _connected=true) và first video frame (set _videoReady=true). Audio chunks từ ElevenLabs đến trong window này được route vào Simli WS nhưng WebRTC audio track chưa fully attached → drop silent.

T+0ms → Simli start() · createPeerConnection · ICE gathering
T+820ms → WS connect · START message received
T+820ms → _markConnected() · isConnected = true
T+830ms → waitUntilVideoReady() resolves (early — only checks _connected)
T+850ms → ElevenLabs WS connect · conversation_initiation_metadata
T+870ms → ElevenLabs audio chunk #1 arrives
T+870ms → Check _simli?.isConnected == true → route to Simli
T+870ms → _simli.sendAudio(chunk) ✗ WebRTC audio track NOT attached → DROP
T+1240ms → First video frame rendered · _markVideoReady()
T+1240ms → Audio track attached · subsequent chunks play ✓
T+1240ms → Learner đã miss ~370ms first message audio
FIX: gate trên isVideoReady, buffer chunks vào queue khi chưa ready, flush khi sẵn sàng. Fallback timeout 1500ms → flush local PcmAudioPlayer.

Interaction rules · prompt card

  • Default state: card mở rộng (expanded) trong 8s đầu mỗi session để learner đọc đề.
  • Auto-collapse: sau 8s không tap → smooth animation về mini-pill (200ms ease-out, scale + opacity).
  • Pulse on new question: khi onAgentResponseComplete fire (examiner kết thúc câu hỏi mới) → pulse subtle 1.5s (scale 1 → 1.04 → 1) báo hiệu "có câu mới, tap để đọc đề".
  • Tap behavior: tap mini-pill → expand full · tap header expanded → collapse. Animation 200ms.
  • Position: bottom — đặt trên controls bar (timer + waveform + End). Tránh che mặt examiner. Transcript overlay đẩy lên trên.
  • Choice variant: card body chỉ hiện nội dung option đã chọn (title + content), không list 3 options. Selected option chip vẫn ở top-right.
  • Reduced motion: nếu OS setting bật → bỏ pulse, expand/collapse instant.
  • Touch target: cả card là touch zone, min height 44pt khi collapsed.

Accessibility · pre-delivery checklist

  • Prompt card text contrast ≥4.5:1 · cream bg #FBF3E7 + ink #1B1B1B = 13.8:1 ✓
  • Status pill background opacity 0.45 + backdrop-blur → contrast over avatar ≥4.5:1 verified
  • End button label "Kết thúc" + icon · không icon-only · accessibilityLabel set
  • Mini-pill có aria-label "Xem đề bài" · screen reader announces khi pulse
  • Touch target tối thiểu 44×44pt cho mini-pill, end button (54pt), waveform area
  • Dynamic Type: prompt card body font-size hỗ trợ scale lên 1.3x không truncate
  • Reduced motion: tắt waveform animation + pulse + expand/collapse animation
  • Safe area: prompt card top = safe-area-top + 64px (dưới status pill); controls bottom = safe-area-bottom + 16px
  • Avatar không phải nguồn thông tin duy nhất · prompt + transcript đảm bảo accessible

Implementation summary

🐛 Bug fix (Direction A + C)

  • interview_session_screen.dart: thêm List<Uint8List> _pendingAgentChunks
  • Đổi gate _simli?.isConnected → _simli?.isVideoReady trong shouldPlayInterviewAudioLocally
  • Khi chunk đến và !isVideoReady: _pendingAgentChunks.add(chunk)
  • Subscribe simli.onVideoReady → flush queue đồng loạt vào _simli.sendAudio()
  • Timer fallback 1500ms: nếu vẫn không ready → flush vào _audioPlayer.addChunk() + set _useSimliAudio=false
  • SimliSessionManager: optional setInputAudioFormat(String?) để Simli match ElevenLabs sample rate

  • _PromptCard widget: AnimatedContainer · expanded ⇄ mini · 200ms ease-out
