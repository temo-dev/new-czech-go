07 · States & Annotations
Các trạng thái và ghi chú kỹ thuật
Session Status States
Đang kết nối
Lấy ephemeral token + WS handshake
Sẵn sàng
Connected, avatar idle
Examiner đang nói
Simli audio → avatar lip-sync + ring pulse
Đang lắng nghe
Mic active → waveform bars animate
Transcript Toggle (CMS-controlled)
show_transcript = true
Overlay hiện cuối session screen. Max 2 dòng, auto-clear sau 4s. Speaker label trên (Examiner / Bạn). Font 12px white 85% opacity.
show_transcript = false
Overlay ẩn hoàn toàn. Screen chỉ có avatar + status pill + controls. Tập trung vào hội thoại thuần.
Accessibility & Touch
· "Kết thúc" button: min 44×44pt, center-bottom, safe area aware
· Choice cards: min 44×44pt tap target, selected state có border + bg change
· VoiceOver label cho status pill: "Trạng thái: examiner đang nói"
· "Kết thúc" → confirmation dialog trước khi end session sớm
Error States
· Token request fail → toast "Không kết nối được, thử lại" + retry button
· WS disconnect mid-session → auto-reconnect 3 lần, sau đó end session và offer save transcript
· Scoring timeout (>2 min) → hiển thị transcript bất kể score
· Mic permission denied → alert với link vào Settings
