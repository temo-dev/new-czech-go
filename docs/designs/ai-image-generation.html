1. Requirement Clarification

Dimension Decision Rationale
Trigger Nút ✨ Tạo bằng AI ngay cạnh nút upload ảnh Không thay đổi flow upload hiện tại — bổ sung path song song
Input Inline collapsible textarea — xuất hiện ngay bên dưới, không mở modal Modal ngắt flow admin; inline nhanh hơn, ngữ cảnh rõ hơn
Provider Flux.1-schnell qua Replicate API ~3s/ảnh, $0.003, không cần key mới nếu đã dùng Replicate
Post-gen Preview thumbnail → admin xác nhận → upload vào asset store Admin kiểm soát, tránh lãng phí storage với ảnh tệ
Scope 5 vị trí (xem Placement) Component <AiImageButton> dùng chung, không duplicate
Rate limit 5 req/phút per admin Tránh spam Replicate API, kiểm soát cost
Timeout 30s, hiển thị error + retry Flux schnell thường ≤5s; 30s là buffer an toàn

5. Component API

Prop Type Required Description
onAssetCreated (assetId: string) => void ✓ Callback sau khi ảnh upload thành công. Parent nhận asset_id để gắn vào form.
disabled boolean — True khi exercise chưa có ID (chưa save). Nút bị dim + tooltip "Lưu bài tập trước".
existingAssetId string | undefined — Nếu đã có ảnh, nút đổi thành "✨ Tạo lại bằng AI". Không ảnh hưởng logic.
initialPrompt string — Prompt mặc định điền sẵn. Optional — context-aware suggestion từ parent.
State Value UI hiển thị
idle Mặc định Nút "✨ Tạo bằng AI" — panel ẩn
open Sau click nút Prompt textarea + Generate button hiện ra (slideDown 180ms)
generating Đang gọi API Progress bar + spinner, textarea disabled, cả hai nút bị dim
preview API trả thành công Thumbnail 120×80px + "Dùng ảnh này" / "Thử lại" / "Hủy"
uploading Sau confirm "Dùng ảnh này" thành spinner, wait for asset_id
error API lỗi / timeout Error banner + prompt cũ còn nguyên + "Thử lại"

6. Backend Endpoint

Field Value
Method + Path POST /api/admin/ai/generate-image
Auth Cookie admin_token (same as all admin endpoints)
Request body { "prompt": string } · max 500 chars
Response (success) { "data": { "asset_id": string, "preview_url": string } }
Response (error) { "error": { "message": string } }
Flow nội bộ 1. Validate prompt → 2. POST Replicate flux-schnell → 3. Poll until done → 4. Download bytes → 5. Lưu asset store (LocalFile/S3) → 6. Trả asset_id + signed preview_url
Timeout 30s total (Replicate thường ≤8s với schnell)
Rate limit 5 req/phút per admin (in-memory counter, reset mỗi phút)
Env var mới REPLICATE_API_KEY
Image format JPEG 512×512 (Flux schnell default) — đủ cho exercise context

7. UX Guidelines

⚡
Disabled khi chưa save (progressive disclosure)
Nút AI dim opacity:0.5 + cursor:not-allowed khi !editingId. Tooltip "Lưu bài tập trước rồi tạo ảnh AI". Cùng pattern với nút Upload hiện tại.
🎬
Animation slideDown 180ms ease-out
Panel mở với transform: translateY(-6px) → 0 + opacity 0 → 1. Dùng @keyframes thuần, không cần thư viện. Respect prefers-reduced-motion.
⌨️
Keyboard: Enter trong textarea không submit
Textarea cho phép xuống dòng tự nhiên. Nút "▶ Tạo" là trigger duy nhất. Tránh accidental submit khi admin đang viết prompt dài.
🔒
Block cả hai nút lúc generating
Khi state === 'generating', cả nút "📁 Tải ảnh lên" và "✨ Tạo bằng AI" đều bị dim + disabled. Tránh race condition nếu admin vô tình nhấn upload đồng thời.
🏷️
AI badge sau khi confirm
Hiển thị pill ✨ AI generated dưới ảnh để admin biết ảnh này từ AI, không phải upload thủ công. Purely informational — không ảnh hưởng logic.
📝
Prompt tiếng Anh
Hint text: "💡 Mô tả bằng tiếng Anh cho kết quả tốt hơn." Flux.1-schnell cho kết quả tốt hơn với EN. Placeholder cung cấp ví dụ thực tế: "A cozy Czech café with a couple having coffee, warm lighting, photorealistic".
💸
Rate limit UX
Khi hit rate limit: error "Đã tạo 5 ảnh trong phút này. Thử lại sau 60 giây." Không cần countdown timer — admin hiểu.

8. Not Doing (V1)

Feature Lý do bỏ
Auto-generate prompt từ nội dung bài Thêm 1 LLM call nữa → +latency +cost; admin viết prompt nhanh hơn
Prompt templates picker (dropdown) Nice-to-have; placeholder example đã đủ cho V1
Batch generate (4 ảnh cùng lúc) Admin cần xem 1 ảnh trước; nhiều ảnh gây confusion
Generation history / gallery Over-engineering V1; admin retry nếu cần
Style preset (realistic / illustration / ...) Admin điều khiển qua prompt text — đủ linh hoạt
DALL-E / Imagen provider Replicate Flux đủ tốt; tránh thêm credential mới
Ảnh > 512px Exercise context không cần ảnh lớn; bandwidth + latency cao hơn không worth
